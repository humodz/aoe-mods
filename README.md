# aoe-mods

This repository contains the metadata for all published mods in the Age of Empires franchise.

https://www.ageofempires.com/mods/

## Updating the metadata files

1. go to: https://www.ageofempires.com/mods/
2. copy the cookies from that website and paste it to cookie.txt in this repo
    - use devtools or the "Cookie Viewer" browser extension
3. run scripts/get-mods.js
